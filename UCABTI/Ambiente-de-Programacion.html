Traductores e Intérpretes UCAB : Ambiente de Programacion
This page last changed on Oct 30, 2007 by carlos.gonzalez.

El ambiente de programación recomendado para este proyecto es el siguiente:

  • Java 5.0
  • ANTLR 2.7.6
  • Eclipse 3.2.1
  • Plugin ANTLR para Eclipse

los estudiantes podrán usar otros ambientes de programación siempre y cuando el generador de compiladores sea ANTLR y el lenguaje de programación sea Java.

Instalación del Ambiente de Programación

  1. Instalar Java 5.0.
  2. Instalar Eclipse 3.2.1 según las instrucciones dadas en su página de download y configuración.
  3. Instalar el Plugin ANTLR para Eclipse según las instrucciones en su sitio Web.
  4. Descomprimir el archivo comprimod con la maqueta de estructura de directorios del proyecto en el disco duro.
  5. Ejecutar Eclipse y usar la opción
    File/Import.../General/Existing Projects into Workspace
    para importar el proyecto a Eclipse.
